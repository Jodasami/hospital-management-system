# Hospital Management System

Sistema web para la administración y gestión de hospitales, médicos y citas médicas.

El sistema permite gestionar información clínica, programar citas y administrar los recursos del hospital mediante una interfaz web.

Este proyecto fue desarrollado como parte de un proyecto académico universitario.

---

## Tecnologías utilizadas

Backend
- ASP.NET
- C#

Frontend
- HTML
- CSS
- JavaScript

Base de datos
- SQL Server
- Stored Procedures

---

## Características principales

- Registro y gestión de hospitales
- Administración de médicos
- Gestión de pacientes
- Programación de citas médicas
- Consulta de historial de citas
- Administración de información médica
